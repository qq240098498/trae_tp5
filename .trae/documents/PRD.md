## 1. 产品概述
宠物喂养寄养后台管理系统，面向宠物服务行业的一站式管理平台，解决寄养计费混乱、服务调度困难、工资核算复杂等问题。

- **主要目的**：统一管理宠物寄养服务、上门喂养服务、饲养员工资核算等核心业务
- **目标用户**：宠物服务店管理员、运营人员、财务人员
- **产品价值**：提升运营效率，降低人工成本，规范业务流程

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 系统管理员 | 系统预置 | 全部功能权限，系统配置 |
| 运营人员 | 管理员创建 | 寄养/喂养服务管理，订单处理 |
| 财务人员 | 管理员创建 | 工资核算、财务报表 |

### 2.2 功能模块
1. **仪表盘：数据概览、快捷入口
2. **寄养管理**：寄养订单、寄养宠物、寄养计费规则
3. **上门喂养**：喂养订单、服务调度、服务记录
4. **饲养员管理**：饲养员信息、考勤记录、工资核算
5. **宠物档案**：宠物信息管理
6. **客户管理**：客户信息
7. **系统设置**：用户管理、权限配置

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 仪表盘 | 数据统计卡片 | 展示今日订单、在寄宠物、待处理任务、月度营收等关键指标 |
| 仪表盘 | 趋势图表 | 近7日订单趋势、服务类型分布 |
| 寄养订单列表 | 搜索筛选 | 按状态/日期/客户筛选 |
| 寄养订单列表 | 订单操作 | 新建、编辑、详情、结算 |

## 3. 核心流程
用户登录 → 查看仪表盘 → 创建寄养/喂养订单 → 分配饲养员 → 服务执行 → 费用结算 → 工资核算

```mermaid
    B --> D["创建上门喂养订单"]
    C --> E["分配饲养员"]
    D --> E
    E --> F["服务执行记录"]
    F --> G["订单费用结算"]
    G --> H["饲养员工资核算"]
    H --> I["工资报表导出"]
```

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深青色 #0F766E（teal-700）作为主色，琥珀色 #F59E0B 作为强调色
- **辅助色**：暖橙色 #FB923C、翡翠绿 #10B981
- **按钮风格**：圆角（rounded-lg），微阴影，hover过渡动画
- **字体**：标题使用思源黑体，正文使用系统无衬线字体
- **布局风格**：左侧导航栏 + 顶部面包屑 + 卡片式内容区
- **图标风格**：Lucide线性图标

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 仪表盘 | 统计卡片 | 渐变背景卡片、数据动画、图标装饰 |
| 寄养订单列表 | 表格 | 斑马纹、状态标签、操作按钮组 |
| 上门喂养 | 日历视图 | 彩色事件块、时间轴 |
| 工资管理 | 核算面板 | 数值高亮、进度条、统计图表 |

### 4.3 响应式
桌面端优先设计，适配1440px标准设计，支持1024px以上完美显示，移动端收缩侧边栏变为抽屉菜单。
